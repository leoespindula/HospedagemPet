Observações

    Para uma boa estadia é necessário trazer:
  • Tapete higiênico, caso o seu pet faça as necessidades dentro de casa;
  • Fralda, caso o seu pet demarque território com frequência;
  • Ração suficiente para os dias que o animal estiver hospedado;
  • Coleira para passear;
  • Cama, brinquedos e demais itens pessoais.
